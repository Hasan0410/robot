# Harry Code - Email for Developers

A modern, responsive landing page for an email delivery service tailored for developers. This project features a stunning 3D robot animation and a sleek, developer-centric design.

## Features

- **3D Robot Animation**: Interactive 3D element powered by [Spline](https://spline.design/).
- **Responsive Design**: Built to look great on all devices.
- **Glassmorphism Effects**: Modern UI with blur and gradient layers.
- **Developer Focus**: Clean typography and clear call-to-actions.

## Technologies Used

- **HTML5**: Semantic structure.
- **CSS3**: Custom styling with gradients and animations.
- **Spline Viewer**: For embedding 3D scenes.
- **Google Fonts**: Modern typography (Outfit, Inter).

## How to Run

Simply open `index.html` in any modern web browser to view the landing page.
